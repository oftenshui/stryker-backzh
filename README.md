# Stryker的安装修复及汉化版
# Stryker Installation, Repair, and Chinese Localization

这是一个基本完成汉化的Stryker，基于Stryker主流版本制作。
This is a Stryker that has been largely localized into Chinese, based on the mainstream version of Stryker.

Stryker是一个非常好的WiFi测试工具，可惜在20260121宣布了服务器停运。
Stryker was a great WiFi testing tool, but unfortunately, its servers were shut down on January 21, 2026.

Stryker服务器停运以后安装、登录、捐赠功能等将无法使用。
After the Stryker server is shut down, installation, login, and donation functions will be unavailable.

但是我们在这里修复了安装功能，现在您只需要下载当前仓库的Release，即可正常使用软件！
However, we've fixed the installation issue here. Now you only need to download the Release version from the current repository to use the software normally.

如果您想使用英文原版，但是却无法下载安装必要文件！您可以先安装我们的汉化版，等待安装好必要文件以后，卸载中文版、安装英文版，即可达到您的目的！
If you want to use the original English version but are unable to download and install the necessary files, you can first install our Chinese version. After the necessary files are installed, uninstall the Chinese version and install the English version to achieve your goal.

- 当Stryker更新的时候，这个仓库可能不会更新
- This repository may not update when Stryker updates.
